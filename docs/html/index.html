VWEP Documentation

Welcome to the documentation for the Vanguard Weapon Base, a framework for creating weapon bases in Garry's Mod. This documentation is a work in progress, so if you see something that's missing or incorrect, feel free to contribute!

Developers

Wether you are a beginner or an experienced developer, you can find all the information you need to get started with VWEP in the Getting Started section. If you are looking for a specific function or hook, you can find them in the API section.
